# Como utilizar ArqueLink
Olá Somos a Equipe N.A.S.A participadora da FLL e nesse README iremos explicar um pouco de como
você iria rodar nosso projeto e mandar para nós os dados do DataBase

---

### Como Rodar o site

* Abra o terminal da pasta Arquelink que esse arquivo está e escreva nele "npx json-server --watch database.json --port 3000"

* Abra o index.html (se o style não estiver funcionando abra pelo Visual Studio Code)

* Depois disso tudo pronto para usar

---

### Como mandar o post

É simples, você precisa apenas:

* Mandar o arquivo "database.json" pelo whatsapp ou email "eqpnasa@gmail.com"
